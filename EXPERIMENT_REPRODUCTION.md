## Commands for Reproducing the Experimental Results

Here we only describe the commands that are only related with *ScalableCA*.

### Setcion 5

#### RQ1 and RQ2

```sh
for benchmark in ` ls cnf_benchmarks `
do
    ./SamplingCA/SamplingCA -input_cnf_path cnf_benchmarks/${benchmark} -output_testcase_path ./SamplingCA/${benchmark}_CA.out
    ./ScalableCA -input_cnf_path cnf_benchmarks/${benchmark} -init_2wiseCA_file_path ./SamplingCA/${benchmark}_CA.out -output_testcase_path ./${benchmark}_CA.out
    rm ./SamplingCA/${benchmark}_CA.out
done
```

#### RQ3

```sh
for benchmark in ` ls cnf_benchmarks `
do
    ./SamplingCA/SamplingCA -input_cnf_path cnf_benchmarks/${benchmark} -output_testcase_path ./SamplingCA/${benchmark}_CA.out
    # Alt-1
    ./ScalableCA -input_cnf_path cnf_benchmarks/${benchmark} -init_2wiseCA_file_path ./SamplingCA/${benchmark}_CA.out -output_testcase_path ./${benchmark}_alt-1_CA.out -use_fid 0
    # Alt-2
    ./ScalableCA -input_cnf_path cnf_benchmarks/${benchmark} -init_2wiseCA_file_path ./SamplingCA/${benchmark}_CA.out -output_testcase_path ./${benchmark}_alt-2_CA.out -use_context_aware
    # Alt-3
    ./ScalableCA -input_cnf_path cnf_benchmarks/${benchmark} -init_2wiseCA_file_path ./SamplingCA/${benchmark}_CA.out -output_testcase_path ./${benchmark}_alt-3_CA.out -use_RALS 0
    # Alt-4
    ./ScalableCA -input_cnf_path cnf_benchmarks/${benchmark} -init_2wiseCA_file_path ./SamplingCA/${benchmark}_CA.out -output_testcase_path ./${benchmark}_alt-4_CA.out -use_RALS 0
    ./CAmpactor/CAmpactor -i cnf_benchmarks/${benchmark} --init_PCA_path ./${benchmark}_alt-4_CA.out -o ./${benchmark}_alt-4_CA.out --twise 3
    # Alt-5
    ./CAmpactor/CAmpactor -i cnf_benchmarks/${benchmark} --init_PCA_path ./SamplingCA/${benchmark}_CA.out -o ./CAmpactor/${benchmark}_CA.out --twise 2
    ./ScalableCA -input_cnf_path cnf_benchmarks/${benchmark} -init_2wiseCA_file_path ./CAmpactor/${benchmark}_CA.out -output_testcase_path ./${benchmark}_alt-5_CA.out

    rm ./SamplingCA/${benchmark}_CA.out
    rm ./CAmpactor/${benchmark}_CA.out
done
```

#### RQ4

```sh
for benchmark in ` ls cnf_benchmarks `
do
    ./SamplingCA/SamplingCA -input_cnf_path cnf_benchmarks/${benchmark} -output_testcase_path ./SamplingCA/${benchmark}_CA.out
    # varying delta
    for delta in 10 50 100 500 1000
    do
        ./ScalableCA -input_cnf_path cnf_benchmarks/${benchmark} -init_2wiseCA_file_path ./SamplingCA/${benchmark}_CA.out -output_testcase_path ./${benchmark}_delta_${delta}_CA.out -delta ${delta}
    done
    # varying L
    for L in 100 250 500 750 1000
    do
        ./ScalableCA -input_cnf_path cnf_benchmarks/${benchmark} -init_2wiseCA_file_path ./SamplingCA/${benchmark}_CA.out -output_testcase_path ./${benchmark}_L_${L}_CA.out -L ${L}
    done
    rm ./SamplingCA/${benchmark}_CA.out
done
```

### Setcion 7

#### Setcion 7.1

```sh
for file in ` ls non_binary_instances/*.model `
do
    filename=$(basename "$file")
    benchmark=$(echo "$filename" | cut -d'.' -f1)
    python3 run.py 1 non_binary_instances/${benchmark}.model non_binary_instances/${benchmark}.constraints ${benchmark}_CA.out
done
```

#### Setcion 7.2

`get_fault.py` is a script that calculates how many faults that a test suite can detect for a given instance. By running the following command, the users can obtain the faults detected by the 3-wise CA generated by ScalableCA.

```
python3 get_fault.py [MODEL_FILE] [TEST_SUITE_FILE] [FAULTS_FILE]
```

```sh
for file in ` ls fault_detection_subjects/*.model `
do
    filename=$(basename "$file")
    benchmark=$(echo "$filename" | cut -d'.' -f1)
    python3 run.py 1 fault_detection_subjects/${benchmark}.model fault_detection_subjects/${benchmark}.constraints ${benchmark}_CA.out
    python3 get_fault.py fault_detection_subjects/${benchmark}.model ${benchmark}_CA.out fault_detection_subjects/${benchmark}_faults.txt
done
```
